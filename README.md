# 🛡️ Sentinela Oculto v3.2 - Detecção Preditiva de Ameaças

Um painel interativo (Single Page Application) desenvolvido para demonstrar a viabilidade arquitetônica e matemática da **Intuição Computacional** na cibersegurança preditiva. 

Este sistema compõe a minha trilha de Projetos em Destaque na formação em Ciência de Dados pela FATEC Cotia. Ele consolida conceitos avançados de inferência estocástica em uma interface web fluida, compartilhando a mesma identidade visual e de usabilidade de outros projetos do portfólio, como o *Wine Analytics Dashboard*.

---

## 🎯 Objetivo do Projeto
Traduzir um modelo conceitual acadêmico em uma prova de conceito funcional. O sistema mitiga a dependência de assinaturas tradicionais (SIEM) ao calcular riscos implícitos em infraestruturas digitais utilizando uma métrica unificada: o **Threat Intuition Score (TIS)**.

## 🚀 Funcionalidades da Aplicação

* **Motor de Inferência (Simulador TIS):** Interface paramétrica para calibrar pesos estocásticos e simular anomalias em tempo real, combinando saídas de algoritmos como *Isolation Forests* e *Redes Bayesianas*.
* **Visualização de Decisão Automatizada:** Integração com gráficos de velocímetro (Gauges) dinâmicos via Plotly, simulando a resposta de um Centro de Operações de Segurança (SOC).
* **Timeline Narrativa (Insider Threat):** Dissecação temporal de um ataque furtivo em 5 dias, demonstrando o encadeamento lógico de estados ocultos via *Hidden Markov Models (HMM)* e retenção de perfil (UEBA).
* **Comparativo de Fadiga Analítica:** Simulação interativa ativando/desativando calibração dinâmica para mitigar *Concept Drift* e reduzir falsos positivos corporativos.

## 🛠️ Stack Tecnológico

* **Linguagem:** Python 3.10
* **Framework Web:** Streamlit
* **Visualização de Dados:** Plotly Graph Objects
* **Computação Científica:** NumPy, Pandas, Scikit-learn
* **Design Pattern:** Arquitetura Modular (Core, Components, Data, Assets) com CSS customizado e injeção de tema nativo (config.toml).
